C++学习笔记（一）：友元函数和友元类

发表于 2017-08-10 | 分类于 C++学习笔记 | 阅读次数

###简介
  C++中存在一种“朋友”关系：如果体现在函数上，我们称之为友元函数，如果体现在类上我们称之为友元类。

###友元函数
  首先，来讨论一下友元函数。 友元函数分为两种：一种为定义在全局的函数，我们称之为友元全局函数，另一种定义在内中，并且将这个函数声明为另外一个类的友元函数，我们就称之为友元成员函数。

#####友元全局函数
先看下面一个例子： 

  class Time
{
    **friend void printTime(Time &t);**
public:

    Time(int hour, int min, int sec);
private:

    int m_iHour;
    int m_iMinute;
    int m_iSecond;
};

Time::Time(int hour, int min, int sec)
{
    m_iHour = hour;
    m_iMinute = min;
    m_iSecond = sec;
}

void printTime(Time &t)
{
    cout << t.m_iHour << ":" << t.m_iMinute << ":" << t.m_iSecond << endl;
}
